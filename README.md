# 🦸‍♀️Smallville City
> “Em um mundo cada vez mais apático, lembrar da esperança nunca foi tão importante.”

## 📖 Sobre o Projeto
Este projeto individual consta com um site desenvolvido com a finalidade de aplicar todos os conhecimentos absorvidos no primeiro semestre do curso de Análise e Desenvolvimento de Sistemas na faculdade SPTech.

## 🎯 Objetivos
- Aplicar conceitos básicos de desenvolvimento web  
- Criar uma interface interativa e temática  
- Utilizar lógica de programação com JavaScript  
- Trabalhar com manipulação de dados (vetores/listas)  
- Integrar cálculos matemáticos ao sistema

## 💻 Tecnologias Utilizadas:
- MySQL Workbench (SQL)
- VSCode (HTML, JavaScript, CSS)
- Oracle Virtual Box (VM)
- Trello
- GitHub
- GitBash
- Power Point (Apresentação)

## ⚙️ Funcionalidades
- Sistema com **cálculos matemáticos**  
- Uso de **vetores (listas)** para armazenar dados  
- Funcionalidade de **sorteio (ex: poderes do Superman)**  
- Interface temática inspirada no universo do Superman
